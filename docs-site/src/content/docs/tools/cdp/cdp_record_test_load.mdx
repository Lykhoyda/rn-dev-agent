---
title: "cdp_record_test_load"
description: "Restore a previously-saved recording from <projectRoot>/"
---

Restore a previously-saved recording from &lt;projectRoot>/.rn-agent/recordings/. Replaces any in-memory events. After loading, call cdp_record_test_generate to render in any format.

## Parameters

| Name | Type | Required | Default | Constraints | Description |
|------|------|----------|---------|-------------|-------------|
| `filename` | `string` | Yes |  | min: 1 | Recording name (without .json) |

## Usage

```
cdp_record_test_load(filename: <string>)
```
