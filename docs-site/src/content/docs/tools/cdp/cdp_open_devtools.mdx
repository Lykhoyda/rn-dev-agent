Report the React Native DevTools frontend URL for the live app + start a multiplexer proxy so DevTools can coexist with the MCP session on RN &lt; 0.85 (RN >= 0.85 uses native multi-debugger). The proxy auto-resumes across reconnects. Returns \{ devtoolsUrl, inspectorWsUrl, hermesWsUrl, mode: "native" | "proxy-active", proxyPort, supportsMultipleDebuggers, rnVersion, guidance \}.

This tool takes no parameters.

## Usage

```
cdp_open_devtools()
```
