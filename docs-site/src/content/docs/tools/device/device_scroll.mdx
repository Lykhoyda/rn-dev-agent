Scroll the screen in a direction. Smoother than device_swipe for list scrolling. Requires an open session.

## Parameters

| Name | Type | Required | Default | Constraints | Description |
|------|------|----------|---------|-------------|-------------|
| `direction` | `enum: up | down | left | right` | Yes |  |  | Scroll direction |
| `amount` | `number` | No |  | min: 0, max: 1 | Scroll amount 0-1 (default ~0.5). 1 = full screen height/width. |

## Usage

```
device_scroll(direction: <enum: up | down | left | right>)
```
