Debug the current screen on the simulator/emulator.

1. Take a screenshot and check cdp_status
2. Gather component tree, errors, console, network logs, and store state
3. Identify any issues (errors, wrong state, missing data)
4. Suggest or apply fixes
5. Verify the fix works
