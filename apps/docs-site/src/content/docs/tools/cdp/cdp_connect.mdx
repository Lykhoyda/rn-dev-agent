Connect only the exact app target on the authority-bound Metro and commit its signed initial-bundle handshake. Omitted port/platform/app values come from the session; conflicts refuse.

## Parameters

| Name | Type | Required | Default | Constraints | Description |
|------|------|----------|---------|-------------|-------------|
| `metroPort` | `unknown` | No |  |  | Must equal the authority-bound Metro port; omitted uses that exact port |
| `platform` | `unknown` | No |  |  |  |
| `targetId` | `unknown` | No |  |  |  |
| `bundleId` | `unknown` | No |  |  |  |
| `force` | `unknown` | No | `false` |  |  |

## Usage

```
cdp_connect()
```
