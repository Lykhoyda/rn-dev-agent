Tap an OS-level dismiss button through the capability-bound runner on the exact session device.

## Parameters

| Name | Type | Required | Default | Constraints | Description |
|------|------|----------|---------|-------------|-------------|
| `label` | `unknown` | No |  |  |  |
| `platform` | `unknown` | No |  |  | Authority-bound platform; conflicting values are refused |
| `timeoutMs` | `unknown` | No |  | min: 1000, max: 60000, integer | Maestro invocation timeout (default 15000ms). |

## Usage

```
device_dismiss_system_dialog()
```
