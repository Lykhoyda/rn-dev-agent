---
title: "cdp_record_test_save"
description: "Persist current recording events to <projectRoot>/"
---

Persist current recording events to &lt;projectRoot&gt;/.rn-agent/recordings/&lt;filename&gt;.json. Filename is sanitized (only [a-zA-Z0-9_-] kept). Use cdp_record_test_load to restore later for re-generation in a different format.

## Parameters

| Name | Type | Required | Default | Constraints | Description |
|------|------|----------|---------|-------------|-------------|
| `filename` | `string` | Yes |  | min: 1 | Recording name (without .json — sanitized) |

## Usage

```
cdp_record_test_save(filename: <string>)
```
