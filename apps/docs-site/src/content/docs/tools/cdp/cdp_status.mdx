Passively report the current authority session, Metro client, and CDP target without connecting, relaunching, dismissing UI, or choosing an ambient target.

## Parameters

| Name | Type | Required | Default | Constraints | Description |
|------|------|----------|---------|-------------|-------------|
| `metroPort` | `unknown` | No |  |  | Diagnostic comparison only; cdp_status never changes the active Metro port |
| `platform` | `unknown` | No |  |  |  |
| `resetArbiter` | `unknown` | No |  |  |  |

## Usage

```
cdp_status()
```
